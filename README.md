# Student Academic Performance Prediction

## Overview

This project predicts student academic performance using Machine Learning techniques.
The system analyzes factors such as:

* Study Time
* Previous Grades
* Absences
* Failures

and predicts:

* Final Grade
* Pass/Fail Status
* Probability of Success

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Models

* Linear Regression
* Logistic Regression

## Features

* Data Cleaning
* Feature Engineering
* Exploratory Data Analysis
* Grade Prediction
* Pass/Fail Classification
* Confusion Matrix Visualization
